5. Please provide a brief description of what your project does and how to use the application. Make sure to include any required information like a username and password.
BuddyLoan: Allows to add buddies using a pin, every buddy should be a registered user and they must share they pin, which is a unique number, in order to add each other.

The application has this modules:
Login: All users must login before use the app, this is the default page if you are not logged in.

Admin: to change basic user info and the password, but not the email which is the username

Buddies: Here you will see all your buddies and a Balance of transactions. This is the place to add buddies, also to delete them.

Transactions: Here you will see all transactions and you can add, edit or delete them.

Dashboard: This is the default page it you are logged in, it presents a Summary of your debt status with your buddies and a general balance.
